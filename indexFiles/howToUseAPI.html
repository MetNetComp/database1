MetNetComp API

How to use API.

By specifying a model and a target metabolite using the following API, you can download a CSV file containing a list of gene deletion strategies.

URL form

https://metnetcomp.github.io/database1/csv/<model>/<target metabolite>.csv

<model>
list of models

<target metabolite>
list of target metabolites

The CSV file can only be downloaded when gene deletion strategies have been determined.

Example

https://metnetcomp.github.io/database1/csv/e_coli_core&succ_e.csv

https://metnetcomp.github.io/database1/csv/iML1515&btn_c.csv
