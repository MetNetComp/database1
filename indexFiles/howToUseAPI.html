MetNetComp API

How to use API.

By specifying a model and a target metabolite using the following API, you can download a CSV file containing a list of gene deletion strategies.

URL form
CSV files for minimal gene deletion strategies
https://metnetcomp.github.io/database1/csv/<model>&<target metabolite>.csv

CSV files for maximal gene deletion strategies
https://metnetcomp.github.io/database1/csv/<model>&<target metabolite>&core.csv

CSV files for equivalent genes
https://metnetcomp.github.io/database1/csv/<model>&equivalentGenes.csv

JSON files for reaction rates for minimal gene deletion strategies
https://metnetcomp.github.io/database1/json/<model>&<target metabolite>&minimal&<id>.json

JSON files for reaction rates for maximal gene deletion strategies
https://metnetcomp.github.io/database1/json/<model>&<target metabolite>&core.json

<model>
list of models

<target metabolite>
list of target metabolites

The CSV file can only be downloaded when gene deletion strategies have been determined.
